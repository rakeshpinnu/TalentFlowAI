# Uipath Data Service

This folder contains the exported UiPath Data Service schema required by TalentFlowAI.

## File

### DataServiceSchema.json

This JSON file includes all the Data Service components required to run the solution, including:

- Entities (Data Tables)
- Choice Sets

## Import Instructions

1. Open **UiPath Automation Cloud**.
2. Navigate to **Data Service**.
3. Click **Import Schema**.
4. Select `DataServiceSchema.json`.
5. Verify that all entities and choice sets are imported successfully before running the solution.

> **Note:** This schema contains all Data Service components used by **TalentFlowAI**. No additional Data Service configuration is required after importing this file.
