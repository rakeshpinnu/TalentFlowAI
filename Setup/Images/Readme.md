The images illustrate each installation and configuration step, including:

- Solution import
- Integration Service connections
- Gmail configuration
- Data Service configuration
- GenAI Activities configuration
- Calendly setup
- Webhook creation
- Data Service schema import
- Storage Bucket creation
- Vector Index creation
- TalentFlowAI-Indexer execution
- TalentFlowAI execution

These images are intended to help users follow the installation guide visually.
